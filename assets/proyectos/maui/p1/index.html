Proyecto Ahorcado (Hangman)

Creado en .Net Maui

.NET MAUI

Proyecto

El objetivo de este proyecto es desarrollar el juego del Ahorcado (Hangman) utilizando .NET MAUI, aplicando conocimientos aprendidos en el curso para el desarrollo de interfaces y de la arquitectura de datos en aplicaciones. Entre los principales temas que se abordan se está la creación y el uso de Layouts en XAML, la configuración adecuada del BindingContext y la implementación de la interfaz INotifyPropertyChanged para manejar la comunicación entre la lógica y la interfaz del XAML.

Retos

Generar botones dinámicos

Crear los botones de forma dinámica para que se ajusten a las diferentes pantallas de los dispotivos.

Utilizar FlexLayout

Implementar un FlexLayout que permita organizar y adaptar los elementos en pantalla de manera flexible.

Uso de BindingContext

Establecer y actualizar un BindingContext que sincronice los datos y la lógica del juego con la interfaz, asegurando que los cambios en el FlexLayout o en cualquier otro control se reflejen de inmediato.
